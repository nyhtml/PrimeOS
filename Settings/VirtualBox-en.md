## PrimeOS in VirtualBox

PrimeOS running inside Oracle VM VirtualBox with the following configuration:

- **Name:** PrimeOS  
- **OS Type:** Windows 10 (64-bit)  
- **Base Memory:** 2048 MB  
- **Video Memory:** 128 MB  
- **Graphics Controller:** VBoxSVGA  
- **Storage:**
  - **SATA Port 0:** PrimeOS.vdi (64.00 GB, Normal)
  - **SATA Port 1 (Optical):** primeos_mainline_0.4.5.iso (1.07 GB)
- **Boot Order:** Floppy, Optical, Hard Disk
- **Acceleration:** VT-x/AMD-V, Nested Paging, Hyper-V Paravirtualization
